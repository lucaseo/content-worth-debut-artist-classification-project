### 기능 :
신인 뮤지션의 데뷔앨범에 대한 웹 매거진 콘텐츠를 제작 여부를 예측

### 기대효과 :
- 에디터들의 컨텐츠 제작 대상 뮤지션을 선정하는 의사결정 서포트
- 신인 뮤지션에 대한 우선순위 선정으로 리서치 시간을 단축
